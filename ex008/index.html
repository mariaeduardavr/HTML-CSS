Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag B. A tag B é uma tag não semântica. Ou seja, ela possui forma, mas não possui significado.

Nesta frase temos um termo em destaque usando a tag strong. A tag strong é uma tag semântica, pois além da forma, ela também dá sentido.

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag I, que é uma tag não semântica.

Nesta frase, temos um termo com ênfase usando a tag EM, que é uma tag semântica.

Texto marcado

Podemos criar, também, texto marcado usando a tag MARK.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno, usando, respectivamente, as tags BIG e SMALL.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado. Para tal, utilizamos a tag DEL.

Texto inserido

Podemos marcar um texto como inserido para indicar que ele foi adicionado posteriormente. Para tal, utilizamos a tag INS.

Existe, também, o texto sublinhado, com a tag U, que é uma tag não semântica.

Texto sobrescrito

Para inserir coisas como x20+3, utilizamos a tag SUP.

Texto subscrito

Para inserir coisas como H20, utilizamos a tag SUB.
